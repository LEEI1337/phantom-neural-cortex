# LAYER 4 - Lazy Bird Automation

**Erbt von:** LAYER-0 + LAYER-1 + LAYER-2 + LAYER-3
**Gilt für:** Autonome Issue → PR Workflows

## Lazy Bird Regeln
- Automatische Projekt-Slot-Zuweisung (A/B/C)
- Projekt-Typ aus Labels erkennen
- Agent-Auswahl basierend auf Issue Labels
- Standard-Projektstruktur anlegen
- GitHub Issue Commenting für Status Updates

## Issue Label → Projekt Typ
- `python` → Python Projekt
- `typescript` → TypeScript Projekt
- `react` → React App

## Issue Label → Agent
- `security`/`architecture` → Claude ($20)
- `documentation`/`bulk-refactor` → Gemini (FREE)
- `github-workflow`/`ci-cd` → Copilot
